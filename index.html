Endoscopía Digestiva Alta

¿Por qué realizar este estudio?

Sirve para diagnosticar y tratar enfermedades del esófago, estómago y duodeno. Permite tomar biopsias sin dolor y aplicar tratamientos como detener hemorragias, tratar várices o extraer cuerpos extraños.

¿En qué consiste?

Se introduce un tubo flexible con cámara por la boca para explorar el tracto digestivo superior.

¿Necesita preparación?

Solo requiere ayuno.

¿Lleva anestesia?

No es doloroso. Puede realizarse con anestesia local en la garganta o sedación intravenosa.

×

Colonoscopía

¿Por qué realizar este estudio?

Es el método más efectivo para detectar pólipos y prevenir el cáncer colorrectal. Permite observar todo el colon, tomar biopsias y realizar tratamientos como la extracción de pólipos.

¿En qué consiste?

Se introduce un tubo flexible por el ano para explorar el intestino grueso. El estudio dura unos 20 minutos.

¿Necesita preparación?

Sí, requiere una limpieza previa del intestino. Ver preparación

¿Lleva anestesia?

Se realiza con sedación. Se solicitan estudios prequirúrgicos y es necesario asistir con acompañante.

Extracción de Pólipos

¿Por qué realizar este estudio?

Es la resección de un pólipo para prevenir o detectar precozmente cáncer. El tejido se analiza para descartar células malignas.

¿En qué consiste?

Se utiliza un lazo de alambre que corta el pólipo mediante corriente eléctrica, a través del endoscopio.

¿Necesita preparación?

Sí. Depende si es por vía alta (ayuno) o baja (preparación similar a la colonoscopía). Ver preparación

¿Lleva anestesia?

Se realiza con sedación. Requiere estudios prequirúrgicos y venir acompañado.

Test de Aire Espirado

¿Por qué realizar este estudio?

Sirve para detectar el sobrecrecimiento bacteriano en el intestino delgado (SIBO), una causa frecuente de síntomas digestivos como hinchazón, gases, diarrea o dolor abdominal.

¿En qué consiste?

El paciente sopla en intervalos regulares dentro de tubos especiales después de ingerir una solución (generalmente lactulosa o glucosa). Se analizan los gases eliminados para identificar fermentación anormal.

¿Necesita preparación?

Sí, requiere una dieta especial los días previos y ayuno antes del estudio. Ver preparación

¿Lleva anestesia?

No. Es un estudio no invasivo, simple e indoloro.
